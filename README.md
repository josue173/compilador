## Run Locally

Download Python

```bash
  https://www.python.org/downloads/
```

Clone the project

```bash
  git clone https://github.com/josue173/compilador.git
```

Go to the project directory

```bash
  cd compilador
```

Install dependencies

```bash
  pip install ply
```

Execute

```bash
  python texto_prueba.py
```
